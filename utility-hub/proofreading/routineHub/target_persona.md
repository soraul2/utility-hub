## III. 일과 패턴 (Typical Day)

```
06:00~06:30  기상 / 스트레칭 / 세면
06:30~07:30  아침 식사 + 출근 준비 (여유가 있으면 독서 30분)
07:30~08:30  출근 (지하철에서 뉴스/SNS)
09:00~12:00  오전 업무 집중 (가장 생산적인 시간)
12:00~13:00  점심 + 동료와 산책
13:00~15:00  회의 / 협업
15:00~17:00  오후 집중 업무 (집중력 저하 시작)
17:00~18:00  업무 마무리 + 내일 준비
18:00~19:00  퇴근
19:00~20:00  저녁 식사 + 휴식
20:00~21:30  자기계발 (운동 or 공부 or 사이드 프로젝트)
22:00~22:30  하루 회고 + 내일 계획 (이상적이나 잘 안 됨)
22:30~23:30  자유 시간 + 취침
```

## IV. 핵심 니즈 (Jobs to be Done)

### Must-Have (없으면 안 쓴다)
1. **"출근 전에 오늘 할 일을 빠르게 정리하고 싶다"**
   → 기본 템플릿 + 빠른 태스크 추가
2. **"업무 중에 태스크 체크로 성취감을 느끼고 싶다"**
   → 포커스 카드 + 실행 타임라인
3. **"퇴근 후 하루를 돌아보며 내일을 준비하고 싶다"**
   → 회고 (KPT) + 내일 계획 연결

### Should-Have (있으면 좋다)
4. **"주말에는 쉬면서도 생산적인 느낌을 받고 싶다"**
   → 휴식일 템플릿 + 주간 리뷰
5. **"나만의 루틴 패턴을 만들고 반복하고 싶다"**
   → 템플릿 저장 + 월간 배치 적용
6. **"한 달 동안 얼마나 성장했는지 한눈에 보고 싶다"**
   → 월간 캘린더 + 완료율 시각화

### Nice-to-Have (차별화)
7. **"내 Google Calendar와 연동되면 좋겠다"**
   → 캘린더 동기화 (향후)
8. **"손으로 쓴 계획표를 사진으로 찍으면 자동 입력되면 좋겠다"**
   → AI 사진 인식 (향후)

## V. 경쟁 앱 대비 포지셔닝

| 경쟁 앱 | 강점 | 약점 (Routine Hub 기회) |
|---------|------|------------------------|
| Todoist | 태스크 관리 완성도, 크로스플랫폼 | 회고/리뷰 없음, 시간 블록 약함 |
| TickTick | 포모도로 내장, 습관 트래커 | 한국어 UX 약함, 루틴 전용이 아님 |
| Notion | 자유도 높음, 템플릿 풍부 | 설정 복잡, 실행 추적 안 됨 |
| Structured | 시간 블록 특화, 깔끔한 UI | 회고 없음, 주간/월간 분석 없음 |
| Google Calendar | 일정 관리 표준 | 계획+실행+회고 통합 불가 |

### Routine Hub의 포지션

```
"계획만 세우는 앱" ──────── Routine Hub ──────── "분석만 하는 앱"
   Todoist, Notion         계획 + 실행 + 회고       Toggl, RescueTime
                           통합 루틴 관리
```

**핵심 차별점**:
- 계획 → 실행 → 회고의 **전체 사이클**을 하나의 앱에서 완결
- Kinetic Timeline으로 **시각적 시간 관리**
- KPT 프레임워크 기반 **구조화된 회고**
- 20~30대 한국 직장인에 최적화된 **한국어 UX**

## VI. 기본 제공 템플릿 연결

페르소나의 일과 패턴에서 직접 도출한 4개 기본 템플릿:

| 템플릿 | 타겟 상황 | 페르소나 니즈 |
|--------|-----------|---------------|
| 출근 루틴 | 평일 기본 하루 | "출근 전에 빠르게 계획 세우기" |
| 자기계발 루틴 | 퇴근 후 성장에 집중하는 날 | "운동+공부+회고를 루틴화" |
| 업무 집중 루틴 | 프로젝트 마감 등 업무 집중일 | "업무 생산성 극대화" |
| 휴식일 | 주말/연차 | "쉬면서도 생산적인 느낌" |

### 기대 효과
- 첫 진입 시 빈 화면 → 4개 템플릿으로 즉시 시작 가능
- "이건 나를 위한 앱이다"라는 감정적 연결
- 첫 세션 이탈률 60~70% → 30~40% 감소 예상
