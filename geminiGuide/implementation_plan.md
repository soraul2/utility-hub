# 구현 계획 - 멀칭 비닐 계산기 UI 최적화 (v0.2.1)

## 목표 설명
사용자가 제공한 **레퍼런스 이미지(디자인 시안)**를 바탕으로 `MulchingFilm.tsx`의 레이아웃을 다듬습니다. 핵심 목표는 **"스크롤 없이 핵심 기능 사용"**입니다. 이를 위해 여백을 줄이고, 폰트 크기를 최적화하며, 정보의 중요도에 따라 강약 조절을 다시 합니다.

## 사용자 피드백 반영
> [!NOTE]
> **피드백**: "전체적으로 너무 큼", "상세 내역은 작아도 됨", "스크롤 없이 입력/결과/버튼이 보여야 함".
> **대응 전략**: Vertical Compactness(수직 압축)를 최우선으로 합니다.

## 변경 제안

### UI 레이어 (UI Layer)
#### [MODIFY] [MulchingFilm.tsx](file:///c:/AiProject/utility-hub/utility-hub/frontend/src/pages/tools/MulchingFilm.tsx)
-   **레이아웃 여백 축소**:
    -   컨테이너 패딩: `p-8` → `p-6` (모바일 대응 고려).
    -   섹션 간 간격: `space-y-10` → `space-y-4` 또는 `space-y-5`.
-   **결과 섹션 (최상단)**:
    -   **필요 수량**: `text-6xl` → `text-5xl font-extrabold text-blue-600`.
    -   **예상 금액**: `text-4xl` → `text-xl font-bold text-gray-700` (서브 정보로 위계 조정).
-   **상세 내역 (중단)**:
    -   배경: 노란색 유지하되 더 심플하게.
    -   텍스트: `text-xs` 또는 `text-sm`으로 축소 (이미지 참조: "1평 = 3.3m²" 등).
    -   내용: 이미지에 있는 "1. 밭 전체 면적", "2. 롤당 면적", "3. 최종 계산" 포맷으로 정리.
-   **입력 폼 (하단)**:
    -   폰트: `text-xl` → `text-lg`.
    -   라벨: `text-sm font-medium` (회색조).
    -   **Button**: `py-5` → `py-3` 또는 `py-4` (높이 축소).
-   **기록 (최하단)**:
    -   스크롤 아래에 위치하도록 배치 (사용자 요청: "바로 안 보여도 됨").
    -   테이블 형태 그리드 디자인 적용 (이미지 참조).
