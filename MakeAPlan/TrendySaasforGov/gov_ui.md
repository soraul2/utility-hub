🎨 디자인 컨셉: "Trendy SaaS for Gov"
룩앤필(Look & Feel): 노션(Notion)이나 토스(Toss)처럼 깔끔하고 직관적인 B2B SaaS 스타일을 지향합니다. 기존 공공기관의 딱딱한 파란색/회색 톤을 배제합니다.

컬러 팔레트: 신뢰감을 주는 소프트한 민트나 스카이블루를 베이스로, 액션 버튼(저장, 복사)에는 생동감 넘치는 코랄 핑크나 써니 옐로우로 포인트를 줍니다.

비주얼 요소: 둥근 모서리(Rounded Corners)를 적극 활용하고, 사이드바나 배경에 은은한 글래스모피즘(불투명 유리 효과)을 적용하여 현대적인 깊이감을 줍니다.

UX 핵심 (No-Login): 로그인 과정이 없으므로, 사용자가 입력하는 정보가 "안전하게 자동 저장되고 있음"을 시각적으로 계속 안심시켜야 합니다.

🖥️ 3단 레이아웃 구조 (3-Column Layout)
전체 화면은 좌측 네비게이션, 중앙 입력 작업대, 우측 실시간 미리보기로 구성됩니다.

[1열] 좌측 사이드바: 네비게이션 & 저장 상태바 (Glassmorphism 적용)
상단 로고: 트렌디한 아이콘 형태의 앱 로고 (예: G-Mate, 공무원 SNS 비서 등).

메인 메뉴: 직관적인 아이콘 메뉴 (홈, 새 글 작성, 사용 가이드).

하단 저장소 상태 (중요): 로그인이 없으므로 프로필 대신 **'브라우저 저장 상태'**를 보여줍니다.

아이콘: 💾 (플로피 디스크) 또는 ☁️ (클라우드+체크) 아이콘

상태 표시: "현재 브라우저에 설정 저장됨" (초록색 점)

Tip: 이 영역을 클릭하면 설정 내보내기/가져오기 메뉴가 뜨도록 확장 가능.

[2열] 중앙 메인: 스마트 입력 작업대 (Clean & Structured)
가장 많은 활동이 일어나는 핵심 영역입니다. 상단 고정 설정과 하단 가변 입력창으로 나뉩니다.

A. 상단 고정 설정 패널 (Persistent Settings Panel)
매번 입력하지 않고 브라우저에 저장되는 영역임을 명확히 합니다.

헤더 타이틀: "기본 설정 (자동 저장됨 ℹ️)"

ℹ️ 아이콘 툴팁: "입력한 정보는 현재 사용 중인 PC 브라우저에만 저장됩니다."

입력 필드 (카드 형태):

🏷️ 기관명: [ 예: 파주시청년공간 GP1939 ] (입력 시 우측에 ✅ 잠깐 표시됨)

📞 문의처: [ 예: 031-940-5100 ]

초기화 버튼: 우측 상단에 작은 🗑️ 아이콘으로 설정 초기화 기능 제공.

B. 콘텐츠 유형 선택 탭 (Pill-Shaped Tabs)
알약 모양의 모던한 탭으로 작성할 콘텐츠 유형을 선택합니다.

[ 📌 프로그램 모집 (Active) ] [ 💼 정책 공고 ] [ 📣 긴급 공지 ]

C. 가변 입력 폼 (Dynamic Form Area)
선택한 탭에 따라 하단 내용이 바뀝니다. (가장 중요한 '프로그램 모집' 기준 예시)

시즌/정보 선택: 둥근 드롭다운과 버튼으로 선택.

월도 선택 [ 2026년 2월 🔽 ]

카테고리 [ 특강 ] [ 취미 ] [ 커뮤니티 ] (버튼 클릭 방식)

일정 픽커: 모던한 날짜/시간 선택 UI.

📅 예약 오픈일 [ 날짜 선택 ] [ 시간 선택 ]

⚡️ 프로그램 리스트 카드 (핵심 기능):

엑셀처럼 복잡하지 않고, 할 일 목록(To-Do List)처럼 심플하게 구성합니다.

입력 행: [날짜 📅] [시간 ⏰] [프로그램명 입력...] [➕ 추가] 버튼

작성된 리스트: 카드 형태로 쌓임.
